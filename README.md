
# 機器使用記録ガントチャート表示プログラム（Go）


このリポジトリは、機器の使用記録をCSVファイル（デフォルト: usage.csv）から読み込み、テキストベースのガントチャートとして可視化するGo言語プログラムです。

## 特長
- コマンドライン引数でCSVファイルを指定可能（未指定時は `usage.csv` を使用）
- 機器ごとの使用期間をガントチャート風に表示
- 全角文字対応（使用者名が日本語でも桁ずれしません）
- 土曜は青、日曜は赤で色分け（対応端末のみ）
- 使用終了日（end_date）が空欄の場合は「?」で右端まで埋めて表示
- 機器ID昇順、同一IDは使用開始日昇順でソート

## CSVフォーマット
- デフォルトファイル名: `usage.csv`

CSVは以下のカラムを持つ必要があります。

```
id,user_name,begin_date,end_date
```
- `id`: 機器ID（文字列）
- `user_name`: 使用者名（全角可）
- `begin_date`: 使用開始日（例: 2025-09-01）
- `end_date`: 使用終了日（空欄可、未返却など）

## 使い方
1. Go 1.18以降をインストール
2. 機器使用記録のCSVファイルを用意
3. 以下のコマンドで実行

```
go run . [CSVファイル名]
```
※ CSVファイル名を省略した場合は `usage.csv` を読み込みます

## サンプル
```
go run . usage.csv
```
